# Chrome History Tracker

A powerful desktop application that reads your Chrome browsing history with advanced features like search, batch loading, metrics, and logging.

## ✨ Features

- 🔍 **Search History** - Real-time search through your entire browsing history
- 📊 **Batch Loading** - Loads 50 records at a time for smooth performance
- ⏱️ **Performance Metrics** - Tracks load times, searches, and errors
- 📝 **Full Logging** - 5 log levels (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- 💾 **Automatic Database** - Reads Chrome history directly - no manual setup!
- 🚀 **Portable EXE** - No Python installation needed
- 🇵🇰 **Pakistan Time** - UTC+5 timezone support

## 📥 Download

### Option 1: Download EXE (Recommended)
1. Go to [Releases](https://github.com/yusrayasin/HistoryTracker/releases)
2. Download the latest `HistoryTracker.exe`
3. Create a folder and put the EXE inside
4. Double-click to run!
